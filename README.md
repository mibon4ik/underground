# Underground Fitness — Sales Academy

Внутренняя база знаний отдела продаж. React-приложение с полной админ-панелью и локальным хранилищем.

## Структура проекта

```
underground-fitness-academy/
├── index.html                        # Точка входа Vite
├── vite.config.js                    # Конфиг Vite
├── package.json
└── src/
    ├── main.jsx                      # Монтирование React
    ├── App.jsx                       # Главный компонент, роутинг, тема
    ├── data/
    │   └── initialData.js            # Все данные сайта (контент, возражения, CRM и т.д.)
    ├── hooks/
    │   └── useStore.js               # localStorage хук — авто-сохранение всех данных
    ├── components/
    │   ├── Sidebar.jsx               # Сайдбар с навигацией и поиском
    │   └── AdminPanel.jsx            # Полная админ-панель (тема, контент, возражения)
    ├── pages/
    │   ├── HomePage.jsx              # Главная страница
    │   ├── ProductPage.jsx           # Продукт (залы, карты, услуги, финансы, post-sale)
    │   ├── SalesPage.jsx             # 8 этапов консультации + кейсы
    │   ├── CrmPage.jsx               # CRM BIG обучение
    │   ├── ObjectionsPage.jsx        # Работа с возражениями (правило 3Н)
    │   └── MaterialsPage.jsx         # Файлы и материалы
    └── styles/
        └── global.css                # Глобальные стили и CSS-переменные темы
```

## Быстрый старт

```bash
npm install
npm run dev
```

Откройте http://localhost:5173

## Продакшн сборка

```bash
npm run build
npm run preview
```

## Альтернатива — standalone HTML

В корне проекта лежит `underground-fitness-academy.html` — полностью самодостаточный файл,
который работает без установки зависимостей. Просто откройте в браузере.
Требует интернет для загрузки React и Babel с CDN.

## Админ-панель

Открывается тремя способами:
- Кнопка **⚙️ Админ** в левом сайдбаре
- Кнопка **⚙️ Админ** в правом нижнем углу (мобильная версия)
- Горячие клавиши **Ctrl + Shift + A**

### Возможности

| Вкладка | Что редактируется |
|---|---|
| **🎨 Тема** | 13 CSS-переменных цветов через color-picker, шрифт, 4 готовых пресета палитры |
| **🏢 Компания** | Название, описание сайдбара, 4 статистики, ценности компании |
| **🎯 Возражения** | Добавить / удалить / редактировать любое возражение: иконка, категория, принцип, шаги диалога, реплики К/М, совет |
| **💬 Консультация** | Заголовки и описания 8 этапов, кейсы менеджера |
| **📊 CRM** | Этапы воронки BIG, KPI-плитки, обязательные поля при реализации |
| **💾 Данные** | Экспорт JSON, импорт JSON, полный сброс к исходным данным |

Все изменения сохраняются автоматически в `localStorage` браузера при каждом вводе —
перезагрузка страницы не нужна.
